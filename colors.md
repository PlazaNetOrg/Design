# PlazaNet Color Pallete
PlazaNet is built around **friendly blues** and **clean whites** to create a playful yet modern interface inspired by Miiverse.

## Core Pallete
| Name | Hex | Usage | Preview |
|------|-----|-------|---------|
| **Primary Blue** | `#3A9FF1` | Main brand color — used for titles, primary buttons, icons | ![Primary](https://img.shields.io/badge/Primary-%233A9FF1?style=for-the-badge) |
| **Secondary Blue** | `#64B7FF` | Accent & hover color — used for gradients, links, highlights | ![Secondary](https://img.shields.io/badge/Secondary-%2364B7FF?style=for-the-badge) |
| **Blue-tinted White** | `#E6F0FF` | Background color; adds gentle depth without losing brightness | ![Primary](https://img.shields.io/badge/BlueTinted%20White-%23E6F0FF?style=for-the-badge) |
| **White** | `#FFFFFF` | Card surfaces, button text, highlights | ![Primary](https://img.shields.io/badge/White-%23FFFFFF?style=for-the-badge) |
| **Black** | `#333` | Main text and icons | ![Primary](https://img.shields.io/badge/Black-%23333?style=for-the-badge) |

## Color Roles
### Primary Blue
- Used in the logo, titles/headers, action buttons, and interactive accents.
### Secondary Blue
- Used for hover, secondary highlights, and gradients.
### Blue-tinted White
- Main background.
- Creates a cozy, Nintendo-like environment.
### White
- Default color for cards, modals, and other primary content areas.
- Keeps focus on posts, text, and drawings.
### Black
- Primary text color.
- Softer than pure black, ensuring comfortable reading.

## Design Notes
- Rounded corners: **15-25px**

---

> The PlazaNet pallete balances **warm nostalgia** and **modern simplicity** — bright, friendly, and easy on the eyes, just like the communities it's built for.
